1) A camada 2 de enlace recebe dados da primeira camada, a física, controlando o fluxo desses dados antes de passarem para a próxima camada e verifica se eles possuem erros. Os protocolos usados nessa camada são: IEEE802.11 - wi-fi, ATM (Asynchronous Transfer Mode) e PPP(Point-to-Point Protocol). Já a camada 3 de rede organiza e faz a priorização de envio dos dados que vieram da camada anterior, usando o endereçamento de IP. Nessa camada temos o protocolo IP (Internet Protocol) e o ICMP (Internet Control Message Protocol).

2) O Windows é um SO padronizado, além de pago. O linux por ser de código aberto oferece uma variedade enorme de acordo com a necessidade de quem vai utilizar, é bastante personalizável e mais rápido de usar para quem gosta de usar o terminal, o deixa mais convidativo para os desenvolvedores. 3) Um projeto OpenSource basicamente é um projeto onde o código fonte dele é público, qualquer um pode ver e sua distribuição é livre. Empresas podem usar de projetos OpenSource para desenvolver outros programas, mas é preciso tomar cuidado principalmente com a questão da segurança de software, pois a vulnerabilidade em projetos derivados de um OpenSource.
